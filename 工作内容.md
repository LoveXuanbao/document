### 数据资源管理更新

```

# 获取metaspace命名空间
METASPACE_NAMESPACE=`kubectl get pod -A | grep metaspace-backend | awk '{print $1}'`

# 获取metaspace-deployment名称
METASPACE_DEPLOYMENT=`kubectl get deploy -n $METASPACE_NAMESPACE | grep metaspace-backend | awk '{print $1}'`

# 修改metaspace镜像地址，注意替换metaspace-backend-1-14-3-soc=xxxxxxxxxx,把xxxx替换为新镜像地址
kubectl set image deploy $METASPACE_DEPLOYMENT 	metaspace-backend-1-14-3-soc=docker.gridsumdissector.com/library/metaspace-backend:latest-97-s9ce7d8a -n $METASPACE_NAMESPACE --record


# 如果升级失败，回退命令
kubectl rollout undo deploy $METASPACE_DEPLOYMENT  -n $METASPACE_NAMESPACE
```
